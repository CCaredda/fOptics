# Intraoperative Functional Brain Mapping

**fOptics** is a software tool for processing intraoperative RGB video or RGB image data to identify activated functional brain areas during neurosurgical operations.  

Related research papers are available in the `papers/` directory.
If you use this software in your work, please cite the following publication:  **XXXX**


**For research use only — not for diagnostic or clinical use.**

---

## Features
- Automated analysis of intraoperative brain imaging or video data acquired during a task-based paradigm
- Multi-threaded processing for efficient data handling
- Visualization tools for mapping functional areas
- Designed for both clinical and research use

---

## Installation

A pre-built Windows installer is available in the [Releases](../../releases) section.

1. Download **fOptics_Installer.exe**.
2. Run the installer and follow on-screen instructions.
3. After installation, launch **fOptics** from the Start Menu or its installation directory.

## Hardware Requirements
- **Operating System:** Windows 10 or Windows 11 (64-bit)
- **CPU:** 64-bit architecture
- **RAM:** Minimum 16 GB recommended for large video datasets

---

## Usage Notes

To ensure optimal performance:

Close other programs before running **fOptics**.  
The software uses a multi-threaded approach to accelerate data computation. Running multiple heavy programs in parallel can increase memory load and risk RAM saturation.
