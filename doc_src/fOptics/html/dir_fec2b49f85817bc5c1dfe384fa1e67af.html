 acquisition.h

 Implementation of the Independant Component Analysis (ICA) function for a blind source separation problem.

 molarcoefffunctions.h
 Set of functions used to compute matrices of coefficients used in the modified Beer Lambert law. These matrices and then used to get chromophore's concentration changes.
 
 oxygenation.h
 Set of functions used to calculate chromophore's concentration changes using the modified Beer Lambert law and to compute the Pearson correlation coefficient between expected and measure time curves.
 
 P_SPM.h
 This class aims to apply the Statistical Parametric Mapping methodology (fMRI standard for identifying functional brain areas) on images of chromophore's concentration changes. This output of this class is a T-Stat matrix that indicates at which point the measured data reject the null hypothesis (no cortical activation).
 
 PAnalyse.h
 Main class that schedules all signal and image processing classes.
 
 PAutomatic_ROI_Detection.h
 Main class that schedules all signal and image processing classes.
 
 PCoeff.h
 Class that aims to compute matrix of coefficient using the autocalibration method. This matrix is then used to project input data and to thus to get signals directly correlated to the tissue's blood volume changes and the oxy-hemoglobin concentration changes.
 
 PDataExtracting.h
 Class that aims to store a 3D image (N.M pixels .K spectral channels) into an array. This array has dimension P.T.K, with P the number of pixels, T the number of frames of the video and K the number of spectral channels.
 
 PFiltering.h
 Class that aims to apply a temporal filtering of data extracted using the PDataExtracting class.
 
 PModifiedBeerLambertLaw.h
 Class that aimsused to compute matrices of coefficients used in the modified Beer Lambert law. These matrices and then used to get chromophore's concentration changes.
 
 pobject.h
 File containing structures used in the whole project.
 
 polyfit.hpp
 File containing functions to find the coefficients of a polynomial p(x) of degree n that fits the data, p(x(i)) to y(i), in a least squares sense. The result p is a row vector of length n+1 containing the polynomial coefficients in incremental powers.
 
 PPixelWiseMolarCoeff.h
 Class that aims to compute matrices of coefficients used in the modified Beer Lambert law. These matrices are calculated in a pixel-wise manner accordingly to the image segmentation provided by the Clustering classes.
 
 PPixelWiseMolarCoeff_copy.h
 
 PProcessTimes.h
 Class that aims to extract rest and activation time temporal indexes stored in the acquisition info files generated during data acquisition. This class also aims to compute the expected heodynamic response function.
 
 PrealTimeProcess.h
 Class that aims to get cortical activation maps in a real time manner. For more information see this paper: https://www.sciencedirect.com/science/article/abs/pii/S195903182030083X (present in the paper folder)
 
 Statistic_functions.h
 Set of functions used to compute statistical tests (T tests, person correlation ...)
